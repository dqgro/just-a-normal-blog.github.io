backstory

It's july 2021 and i decide to start programming. Prior to this I had little to no programming experience. I use to think that coding was just a hobbby. Something you learn in a few days and get over with real quickly. But in fact, programming is so much more than that. It opens a whole new world of endless possibilities and no limits what so ever. I started learning JavaScirpt through Khan Academy. It started off fun, but little by little it started getting more interesting. Sooner then i thought i was already getting into HTML.

My opinions on HTML and webpage making

Eventually, i started making webpages just like the one you're currently on. My opinion on them are that they are awesome. Think about a site you visit every day. Per say YouTube or even Khan Academy. There are so many things that you can make with HTML and CSS. This webpage that you're on right now is made with HTML. Of course im not the best Programmer but im working on it.

Conclusion

Unfortunatly it takes days, months and years to master HTML and CSS. But sooner or later you'll notice progress. I like saving all my webpages on my computer in a file called "JavaScript" so in a few months or years i can go back and see my progress which i really recommend doing so. With programming you can make amazing things. So, get at it!
